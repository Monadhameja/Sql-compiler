I worked on the Execution and Semantic Phase of the SQL Query Compiler. My role was to evaluate the parsed query (AST) and generate the final output.

⸻
 Key Work
	•	Evaluated the AST generated by the parser
	•	Implemented WHERE condition filtering
	•	Handled logical operators: AND, OR
	•	Supported relational operators: =, >, <, >=, <=
	•	Implemented LIKE using pattern matching (%, _)

⸻
 Working Flow
	1.	Receive AST from parser
	2.	Extract fields, table, and conditions
	3.	Apply filtering logic on dataset
	4.	Evaluate conditions (AND / OR / LIKE)
	5.	Return final result
